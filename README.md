## 安装说明

在线文档参考：[安装说明](https://zh.d2l.ai/chapter_installation/index.html)
以下给出基于conda的安装方法：

1. 创建conda环境 `conda create --name d2l python=3.9 -y`
2. 激活环境 `conda activate d2l`
3. 安装d2l `pip install d2l`
4. 安装torch `pip install torch torchvision`
5. 安装tensorflow `pip install tensorflow tensorflow-probability`
6. 安装d2l `pip install d2l`
